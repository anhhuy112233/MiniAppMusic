# 🎵 Mini Music Player  

## 📌 Giới thiệu  
- Mini Music Player là một ứng dụng nghe nhạc nhỏ gọn, đơn giản nhưng đầy đủ tính năng.  
- Cho phép người dùng phát nhạc từ danh sách nhạc có sẵn.  
- Giao diện trực quan, dễ sử dụng.  

## 🚀 Tính năng  
- 🎼 Phát / Tạm dừng nhạc.  
- ⏭ Chuyển bài / Quay lại bài trước.  
- 🔁 Lặp lại một bài hát hoặc toàn bộ danh sách phát.  
- 🔀 Phát ngẫu nhiên danh sách nhạc.  
- 🔊 Điều chỉnh âm lượng.  
- 🎵 Hiển thị thông tin bài hát đang phát (Tên bài hát, nghệ sĩ).  
- 📂 Hỗ trợ danh sách phát tùy chỉnh.  

## 🛠 Công nghệ sử dụng  
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express (nếu có)  
- **Audio API:** Web Audio API / HTML5 Audio Element
